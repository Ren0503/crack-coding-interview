# 8. Ma trận không

## Mô tả

<p>Viết thuật toán sao cho nếu một phần tử trong ma trận MxN là 0, thì toàn bộ hàng và cột của nó được đặt thành 0.</p>


<p><strong>Ví dụ 1: </strong></p>

<pre>

<strong>Input: </strong>

[

  [1,1,1],

  [1,0,1],

  [1,1,1]

]

<strong>Output: </strong>

[

  [1,0,1],

  [0,0,0],

  [1,0,1]

]

</pre>

<p><strong>Ví dụ 2: </strong></p>

<pre>

<strong>Input: </strong>

[

  [0,1,2,0],

  [3,4,5,2],

  [1,3,1,5]

]

<strong>Output: </strong>

[

  [0,0,0,0],

  [0,4,5,0],

  [0,3,1,0]

]

</pre>
